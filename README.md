📋 Validação de Formulário com JavaScript

Este projeto tem como objetivo demonstrar a implementação de um sistema de validação de formulários utilizando JavaScript puro, garantindo que os dados inseridos pelo usuário estejam corretos antes do envio.

🚀 Funcionalidades
Validação de campos obrigatórios
Verificação de formato de e-mail
Validação de senha (tamanho mínimo, critérios, etc.)
Feedback visual em tempo real para o usuário
Prevenção de envio de formulário com dados inválidos
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript (Vanilla JS)
💡 Como funciona

O sistema captura os dados inseridos pelo usuário e realiza verificações através de funções JavaScript. Caso algum campo esteja inválido, mensagens de erro são exibidas e o envio do formulário é bloqueado até que todas as informações estejam corretas.

📚 Objetivo do projeto

Este projeto foi desenvolvido com foco em aprendizado, visando melhorar habilidades em:

Manipulação do DOM
Tratamento de eventos
Validação de dados no frontend
Boas práticas de usabilidade (UX)

git clone https://github.com/VictorEmanoel97/Validacao_de_Formulario.git
