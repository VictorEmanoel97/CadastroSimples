# 📋 Página de Cadastro com JavaScript

Este projeto tem como objetivo demonstrar a criação de uma página de cadastro completa, utilizando JavaScript puro para validar e processar os dados inseridos pelo usuário antes do envio.

## 🚀 Funcionalidades
* Cadastro de usuário com múltiplos campos
* Validação de campos obrigatórios
* Verificação de formato de e-mail
* Validação de senha (tamanho mínimo, critérios, etc.)
* Confirmação de senha
* Feedback visual em tempo real para o usuário
* Prevenção de envio com dados inválidos
## 🛠️ Tecnologias utilizadas
* HTML5
* CSS3
##JavaScript (Vanilla JS)
💡 Como funciona

A página permite que o usuário preencha seus dados de cadastro, como nome, e-mail e senha.

O sistema utiliza JavaScript para capturar essas informações e realizar validações automaticamente. Caso algum campo esteja incorreto, mensagens de erro são exibidas na tela, impedindo o envio até que todos os dados estejam válidos.

## 📚 Objetivo do projeto

Este projeto foi desenvolvido com foco em aprendizado, visando melhorar habilidades em:

* Manipulação do DOM
* Tratamento de eventos
* Validação de dados no frontend
* Criação de interfaces de cadastro
* Boas práticas de usabilidade (UX)
